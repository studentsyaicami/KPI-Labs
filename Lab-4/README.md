# Лабораторна робота №4

- Варіант: 5
- Звіт: *Report.pdf*

### Завдання:
    1) Агрегатні функції Використано функції SUM, AVG, MIN, MAX, COUNT для отримання загальної статистики продажів (виручка, середня ціна).
    2) Групування (GROUP BY) та сортування (ORDER BY) Створено звіт по покупцях: скільки кожен витратив коштів. Дані відсортовано за спаданням суми.
    3) Використання HAVING Застосовано HAVING для фільтрації згрупованих даних (виведення товарів з продажами більше 2 одиниць).
    4) Порівняння WHERE та HAVING Продемонстровано спільну роботу: WHERE фільтрує дату продажу, а HAVING фільтрує підсумкову суму по групі.
    5) DISTINCT та LIMIT Використано DISTINCT для отримання списку унікальних активних клієнтів та LIMIT для пошуку однієї найбільшої угоди.
